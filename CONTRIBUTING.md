# Contributing

Thanks for your interest in contributing!

## Prerequisites
- Java 21
- Maven 3.8+
- Docker (optional)

## Setup
```bash
mvn clean install
```

## Run
```bash
mvn spring-boot:run
```

## Tests
```bash
mvn test
```

## Style
- Keep changes small and focused.
- Prefer clear naming and short methods.
- Add/adjust tests when behavior changes.

## Commit Messages
Use Conventional Commits:
- feat: new feature
- fix: bug fix
- test: tests only
- chore: tooling or maintenance

## Pull Requests
- Describe what changed and why.
- Include test evidence if applicable.
